Conway's Game of Life

The Game of Life is a cellular automaton devised by the British mathematician John Horton Conway in 1970. The game is a zero-player game, meaning that its evolution is determined by its initial state, requiring no further input. This version was created using p5.js.

Snake

Snake is the common name for a video game concept where the player maneuvers a line which grows in length, with the line itself being a primary obstacle. This version was created using p5.js.

Tic Tac Toe AI

Tic Tac Toe is a game played between two people, where one is a cross and the other is a nought. The objective is to get three of your own shapes in a line. In this version youy play against the computer. The computer makes perfect desicions and is unbeatable. It was created using p5.js.
